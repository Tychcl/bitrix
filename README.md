# Установка и тд
1. скачать https://www.1c-bitrix.ru/download/vmbitrix.php HyperV windows 10
2. в hyperV `Импорт виртуальной машины`
3. когда будет пункт выбора копировать или восстановить, выбираем копировать
4. подключаем switch
5. запускаем виртуалку, заходим под root, пароль пишется у красных рамок рядом
6. меняем пароль, предложит автоматически
7. все, заходим по IP на виртуалке и выполняем установку

# Приколы с кастомными модулями
1. Модуля нет в `Настройки -> настройки продукта -> модули`
   Если модуль в папке `local/modules` стоить проверить `Marketplace -> Установленные решения`   
2. Установка
   Чтобы установить нужно так же перейти в `Marketplace -> Установленные решения` и слева от модуля нажать на 3 полосочки, там должен быть пункт меню - `установить`
