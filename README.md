# Приколы с кастомными модулями
1. Модуля нет в `Настройки -> настройки продукта -> модули`
   Если модуль в папке `local/modules` стоить проверить `Marketplace -> Установленные решения`
2. Установка
   Чтобы установить нужно так же перейти в `Marketplace -> Установленные решения` и слева от модуля нажать на 3 полосочки, там должен быть пункт меню - `установить`
